# Roadmap

## Milestone 1: Bootable Foundation

- Build ArchISO with Vibe branding and installer autostart.
- Complete archinstall profile generation.
- Package `vibectl` and `vibe-plugin-runtime`.
- Ship KDE defaults and Lite/Full mode profiles.

## Milestone 2: Plugin Ecosystem

- Plugin repository index format.
- Plugin signature verification.
- GUI permission prompts.
- Plugin store UI.
- PolicyKit integration for approved privileged APIs.

## Milestone 3: Desktop Polish

- Vibe KDE global theme.
- First-run setup wizard.
- Driver manager UI.
- Backup and restore tool.
- Settings center integration.
